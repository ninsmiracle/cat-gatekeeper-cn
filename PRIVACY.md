# Privacy Policy

Cat Gatekeeper is designed as a local-only browser extension.

## Data Stored

The extension stores these settings in Chrome local storage:

- Whether the extension is enabled.
- The target website list.
- Trigger minutes.
- Break minutes.
- Sound preference.

## Data Collection

Cat Gatekeeper does not collect personal data, browsing history, analytics, cookies, account data, or website content.

## Data Sharing

Cat Gatekeeper does not send data to any server and does not share data with third parties.

## Permissions

The extension runs on `http` and `https` pages so it can show the break overlay on websites selected by the user. It uses local storage to save settings.
